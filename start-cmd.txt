端末へ配布

Generate App Bundles or APKs

これを選択すると、次に「Android App Bundle」か「APK」かを選ぶ画面が出ますので、「APK」 を選択して進めてください。

もしメニュー構成が少し違う場合、もっとも簡単なのは以下の階層を探すことです：

Build メニューを開く
Build Bundle(s) / APK(s) という項目を探す（もしあれば）
その中にある Build APK(s) をクリック
これを実行すると、しばらくして右下に 「APK(s) generated successfully」 と通知が出ます。その中の 「locate」 という青い文字をクリックすると、フォルダが開き、インストール用の app-debug.apk が見つかります。

これをタブレットにコピーして開けば、インストールが始まります！

---
運用時のポイント：
今後、画面のボタンの色を変えたり、新しい機能を追加したりした場合は、以下の「3点セット」を実行して各タブレットに配布してください：

PCでビルド: npm run build
ネイティブ同期: npx cap sync
APK作成: Android Studio で Build APKs を実行して各端末へ配布。


customer-api-prototype ディレクトリはGitリポジトリになっていないため、以下のコマンドを実行してバックエンドをCloudflare Workersへデプロイ

cd customer-api-prototype
npm run deploy


Cloudflare APIへの接続で認証エラー（Authentication error [code: 10000]）が発生した場合、

一度ログアウトしてから再ログインすると正常に動作する場合がある

npx wrangler logout

npx wrangler login

npm run deploy
